# HTML Previewer

A modern, lightweight **HTML Previewer** that lets you write HTML code and instantly preview it across multiple device sizes. Designed with a premium glassmorphism interface, smooth animations, auto-save, and responsive testing tools.


---

## ✨ Features

* 📝 Live HTML editor
* ⚡ Instant preview
* 💾 Automatic local save
* 📱 Responsive device preview
* 🖥️ Full, Desktop, Laptop, Tablet & Mobile modes
* 🔄 Device rotation
* 🔍 Zoom In / Zoom Out
* ⛶ Fullscreen preview
* ♻️ One-click preview refresh
* 📏 Line numbers
* 📊 Character counter
* 🎨 Premium glassmorphism UI
* 🌙 Modern dark theme
* 📂 Default starter HTML template
* 🖥️ Desktop & Tablet optimized
* 📦 No installation required
